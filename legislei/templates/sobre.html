Sobre Legislei

Objetivos do projeto

O projeto Legislei tem como objetivo prover ao cidadão uma ferramenta de acompanhamento parlamentar. O sistema gera relatórios das atividades realizadas pelo político ao longo de um período de tempo e envia para o usuário ao término desse período. Dessa forma, informações de proposições, votações e sessões presentes são entregues ao eleitor.

O sistema é software livre, o que significa que os usuários tem a liberdade de modificar e distribuir o software, desde que mantendo os projetos derivados também livres. Dessa forma, esse servidor foi montado para a conveniência de todos, uma vez que qualquer pessoa pode baixar o código-fonte e rodar em sua própria máquina. Para mais informações em como fazê-lo, consule a documentação.

Casas legislativas disponíveis

Quais são e como acrescentar casas legislativas ao projeto

Atualmente, a Câmara dos Deputados, a Assembleia Legislativa de São Paulo e a Câmara Municipal de São Paulo são as casas legislativas cadastradas no projeto. Porém, como o objetivo do projeto é acompanhar qualquer parlamentar que se deseje, a ambição do sistema é integrar com todas os órgãos legislativos do páis.

Se você for desenvolvedor e quiser acrescentar uma casa ao sistema, consulte a documentação para entender como integrar uma nova legislatura ao projeto. Depois, crie um fork do repositório do sistema no GitHub para desenvolver a extensão. Quando terminar, abra um Pull Request.

Se você não for desenvolvedor ou não tiver interesse em contribuir com o desenvolvimento e gostaria que alguma casa fosse acrescentada ao projeto, abra uma issue no GitHub.
